# 🎮 Pokémon Game

A fun and interactive Pokémon-style game where players can explore, battle, and collect Pokémon!

## 🚀 Features

- 🗺️ Explore different areas/worlds  
- ⚔️ Battle wild Pokémon and trainers  
- 🎒 Catch and collect Pokémon  
- 📈 Level up and evolve Pokémon  
- 💾 Save and load game progress  

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Express (optional)  
- Database: MongoDB / Local Storage
